# AIRAWeb 文档入口

这份文档用于回答三个问题：
- 现在项目已经做到什么程度
- 相对 `main`，当前分支新增了哪些能力
- 本地开发、调试、导入数据应该从哪里开始

## 当前代码状态

当前分支已经从最初的“课程浏览原型”推进到一个可本地完整调试的学习闭环版本，核心能力包括：

- 课程、试卷、题目从数据库读取
- FDS 课程和试卷可通过脚本导入
- 用户注册 / 登录 / 登出可用
- 用户、验证码、access token、refresh token 已落库
- 试卷页支持刷题模式和模拟考模式
- 题目解析支持 Markdown / 数学公式
- 同学解析支持投稿、编辑、投票
- 收藏、错题本、做题记录、个人资料均已接入后端
- 课程简介已从静态占位文案改为数据库字段，并补上“用户提交、管理员审核”的修改链路
- 管理员审核中心已上线，集中处理课程简介、教师信息、评分标准三类待审核提交

## 相对 `main` 的主要增量

今天这批改动相对 `main`，可以按 6 类理解：

### 1. 数据层从“原型内存”走向“可调试数据库版”
- 新增课程导入与试卷导入命令
- 课程、试卷、题目改为落 PostgreSQL
- 收藏、错题本、做题记录、用户资料改为持久化
- 认证从内存用户切换为数据库版用户与 token

### 2. 学习流完成 P0 闭环
- 做题页支持刷题模式
- 做题页支持模拟考模式
- 倒计时、自定义时长、自动交卷 / 超时继续
- 左侧题型目录跳转

### 3. 题解协作能力上线
- 题目自带参考解析
- 用户可以提交自己的题解
- 作者可编辑自己的题解
- 其他用户可点赞 / 点踩 / 撤回
- 前端展示评分最高的 3 条题解

### 4. 个人中心成型
- 个人资料页
- 收藏页
- 错题本页
- 做题记录页
- 头像上传、昵称编辑、等级展示

### 5. 前端体验增强
- 首页从空白跳转页改为真实 landing page
- 顶部导航整理为：首页 / 课程 / 个人中心
- 注册页补了邮箱验证码、密码显隐、小眼睛按钮、字段顺序优化
- 题目支持 Markdown + LaTeX 渲染
- 判断题、填空题、编程题有基础交互

### 6. 本地开发工具完善
- 一键启动前后端脚本
- 自动安装 / 启动 PostgreSQL 的后端脚本
- FDS 测试数据导入脚本

## 本地开发入口

### 启动前后端

```bash
bash scripts/dev.sh
```

### 清库并导入 FDS 测试数据

```bash
bash scripts/test-import.sh
```

### 只导入课程

```bash
bash scripts/import-courses.sh
```

## 当前推荐调试路径

### 用户链路
1. 打开 `/register`
2. 获取开发模式验证码
3. 注册账号
4. 重启后端后再次登录，验证账号仍保留

### 学习链路
1. 打开 `/courses`
2. 搜索 `CS1018F`
3. 进入课程详情和试卷页
4. 先测刷题模式，再测模拟考模式

### 个人中心链路
1. 打开 `/profile`
2. 修改昵称、上传头像
3. 进入 `/profile/favorites`
4. 进入 `/profile/wrongbook`
5. 进入 `/profile/records`

### 课程简介链路
1. 打开 `/courses`
2. 确认课程卡片显示真实课程简介
3. 打开 `/courses/CS1018F`
4. 提交一条课程简介修改建议
5. 打开 `/admin/reviews`
6. 审核通过后，再回到课程广场确认简介已更新

### 管理员链路
1. 运行 `bash scripts/test-import.sh`
2. 使用 `admin / admin@123` 登录
3. 打开 `/admin/reviews`
4. 审核课程简介、教师信息、评分标准三类提交
